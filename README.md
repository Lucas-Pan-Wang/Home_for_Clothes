# Home_for_Clothes
--------------------------------------------------
## 关于库
本文档作为项目实训流程标准化演练中《衣舍》项目的需求书，使该项目更切实有效，并作为后续项目开发，测试，验收的最主要依据文献。
## 项目简介
项目目标：设计打造出一款衣物相关的实用类App,主要功能包括记录衣物存放位置和虚拟衣物穿搭，同时还集社区分享、穿搭日记、衣物寻找于一体。
